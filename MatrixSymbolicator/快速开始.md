# 🚀 Matrix 符号化工具 - 快速开始

一分钟上手使用 Matrix 符号化工具！

## 📦 第一步：打开项目

```bash
cd /Users/momo/Desktop/MatrixTestApp/MatrixSymbolicator
open MatrixSymbolicator.xcodeproj
```

在 Xcode 中按 `⌘R` 运行应用。

## 📤 第二步：上传文件

### 上传 dSYM 符号表

将 dSYM 文件拖到左侧上传区域：

```
matrix-symbolicate-server/dsyms/20260103_180622_MatrixTestApp.dSYM.zip
```

### 上传错误日志

将 JSON 日志拖到左侧上传区域：

```
matrix-symbolicate-server/reports/1767434772633665000_lag_report_1_1767434772.json
```

## ⚡ 第三步：开始符号化

点击 **"开始符号化"** 按钮，等待 2-5 秒。

## ✅ 完成！

## 💡 快捷操作

- **复制结果**: 点击右上角 `📋 复制` 按钮
- **导出文件**: 点击右上角 `📤 导出` 按钮
- **清除文件**: 点击文件右侧的 `❌` 按钮
